## 🚀 Getting Started

 ### ✅ Prerequisites
Make sure you have:
- 🧩 Node.js 18+ (recommended)
- 📦 npm, pnpm, or yarn

### 📥 Install Dependencies
```bash
npm install
# or
pnpm install
# or
yarn
```
### 💻 Run in Development
```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```
### 🏗️ Build for Production
```bash
npm run build
# or
pnpm build
# or
yarn build
```
### 🔍 Preview Production Build
```bash
npm run preview
# or
pnpm preview
# or
yarn preview
```
